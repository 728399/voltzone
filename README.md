# VoltZone — Electronics Store
> Project for Electronic Commerce

A mobile-first online electronics store built with HTML, CSS, and JavaScript.

## Pages
- **Home** — hero banner, product categories, flash sale timer
- **Products** — full catalog with category filters
- **About** — company story, values, team
- **Blog** — tech articles and buying guides
- **Contact** — contact form and store info
- **Terms & Conditions** — payment policy, returns, GDPR

## Features
- Responsive mobile-first design
- Shopping cart with quantity management
- Simulated payment gateway (Card, PayPal, Apple Pay, Google Pay)
- Live countdown timer for flash sales
- Product filtering by category

## Monetization
One-time payments via simulated Stripe checkout.

## Tech Stack
HTML · CSS · JavaScript (vanilla)
